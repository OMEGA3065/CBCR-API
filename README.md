# Component Based Custom Role API (CBCR-API)

An API which adds Custom Role support for LabAPI, using the concepts used in my [Component Based Custom Item API](https://github.com/OMEGA3065/CBCI-API).
You can find examples on how to use the API in the [Examples folder](https://github.com/OMEGA3065/CBCR-API/tree/main/CustomRoleExamples).

# Installation
Install this as any other [LabAPI](https://github.com/northwood-studios/LabAPI) plugin.
Installation guide in the [LabAPI Readme](https://github.com/northwood-studios/LabAPI).

# Developer Package reference
This Library is also available as NuGet package. See: https://www.nuget.org/packages/Omega3065.ComponentBasedCustomRoleAPI/

# Developer Wiki
At this moment the only wiki is viewing the quite well commented code in combination with
looking at the provided [Examples](https://github.com/OMEGA3065/CBCR-API/tree/main/CustomRoleExamples).

# Current Features
- Custom Roles with attachable components
- Support for basic components (role initialization and role given hints)
- Integration with [RaCustomMenu](https://github.com/Bankokwak/RaCustomMenu)
    1. Give custom roles to players using the UI (Dummies Tab in RA)
